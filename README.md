 Student Management System (Flask)

A simple and efficient web-based Attendance Management System built using Flask and SQLite.
This application allows users to register, log in securely, and mark their attendance with real-time timestamps. 
It also provides a personalized dashboard where users can view their attendance history in a structured format.


Features

User Registration & Login System.
Session-based Authentication.
One-click Attendance Marking (Clock-in).
Automatic Timestamp Recording.
Dashboard with Attendance History.
Clean and Simple UI.
